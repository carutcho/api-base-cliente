
# API: Base de clientes - Builders

Api para gestão de clientes da Builders

##Tecnologias:
- Java: 12
- Banco de dados: MySql
- Springboot: 2.7.0
- Lombok

## Documentação

[Documentação](http://localhost:8080/swagger-ui.html)

[Postman](https://www.getpostman.com/collections/ca1adb8a2d9ab266b05c)


## Funcionalidades

- Buscar cliente por id
- Incluir cliente
- Atualizar cliente
- Buscar cliente por filtro paginado


## Apêndice

- A Api está configurada para autenticação Basic para basico de segurança:
Fazer requisições com Authorization Basic User: builders Password: builders123
- A base de dados está configurada com driver de MySql e está para criar incialmente a estrutura via codefirst (hibernate)
